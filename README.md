# 🚀 InterviewIQ - AI Mock Interview Platform

An AI-powered mock interview platform that helps users practice technical and HR interviews with real-time AI-generated questions, resume-based interviews, and detailed performance feedback.

---

# ✨ Features

- 🤖 AI-generated interview questions
- 📄 Resume-based personalized interviews
- 💬 HR & Technical interview modes
- 🎤 Voice-enabled interview support
- 📝 Real-time answer evaluation
- 📊 Detailed performance analysis
- ⭐ Overall interview score
- 📈 Strengths & Weaknesses report
- 💡 Personalized improvement suggestions
- 🔐 Secure authentication

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/fitbitgaming-lgtm/INTERviewiq.git
cd INTERviewiq
```

Install dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd ../server
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file inside the server folder.

```env
MONGODB_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET

GEMINI_API_KEY=YOUR_API_KEY
```

---

# ▶️ Run Project

Backend

```bash
cd server
npm run dev
```

Frontend

```bash
cd client
npm run dev
```
